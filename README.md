# Contractions

## Installation

```shell
python3.12 -m venv venv
. venv/bin/activate
```

```shell
python -m pip install -U pip
python -m pip install -r requirements.txt
```

## Démarrer le serveur

Pour lancer un serveur web accessible à [`http://localhost:2018`](http://localhost:2018) :

```shell
python server.py
```
